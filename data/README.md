# Data Directory

This directory contains all data files for the TransPort-PH project.

## Structure

- **Raw Data**: Original data from various sources (World Bank, DPWH, JICA, etc.)
- **Processed Data**: Cleaned and merged panel datasets
- **Reports**: Data quality and analysis reports

## Data Sources

1. **World Bank** - Economic and infrastructure indicators
2. **DPWH** - Department of Public Works and Highways data
3. **JICA** - MRT/LRT transit data
4. **LTFRB** - Land Transportation Franchising and Regulatory Board
5. **PSA** - Philippine Statistics Authority
6. **SWS** - Social Weather Stations satisfaction surveys
7. **OpenAQ** - Air quality data (PM2.5)
8. **TomTom** - Traffic congestion data
9. **UITP** - Modal share data
10. **ADB** - Asian Development Bank project data

## Notes

- Large data files are excluded from git (see `.gitignore`)
- Run data collection scripts to regenerate raw data
- Processed data is generated by preprocessing pipelines
